### 3.DeLog - 개발자 기술 블로그 플랫폼

`2027.09~2027.11` | 개인 프로젝트

풀스택 역량 향상을 목표로 **프론트엔드부터 백엔드, 배포까지 혼자 구축**한 기술 블로그 플랫폼입니다.
마크다운 에디터 기반 글쓰기, 태그 분류, 댓글, 좋아요 기능을 포함합니다.

<details>
<summary>상세 내용 보기</summary>

**아키텍처 구성:**

```
[Client]Next.js(App Router)
    Rest API
[Server] Node.js + Express
    ORM (Prisma)
[DB] PostgreSQL (Supabase)
[Deploy] Vercel (FE) + Railway (BE)
```

**핵심 구현 사항:**
- Next.js App Router 기반 SSR/SSG 혼합 전략으로 SEO 최적화
- Express 서버에서 JWT 인증 미들웨어 직접 구현(Access / Refresh Token 전략)
- Prisma ORM을 활용한 PostgreSQL 스키마 설계 및 관계형 데이터 모델링
- Supabase Storage를 연동하여 이미지 업로드 기능 구현
- Railway를 통한 백엔드 서버 무중단 배포 환경 구성

**기술 스택:**

`Next.js` `TypeScript` `Node.js` `Express` `Prisma` `PostSQL` `Supabase` `Railway` `Vercel`

</details>
